## Context

Sweden is missing a modern marketplace for used computer parts which DealFortress looks to fix

## Tech stack


  - .NET with C#

  - Entity Framework

  - React with Vite & Typescript

  - Github Actions & Project

  - Azure Sql DB, Azure web app services & Azure static webpages
